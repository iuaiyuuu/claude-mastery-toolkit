# 🚀 Claude Mastery Toolkit

A curated collection of production-grade Claude skills for building real-world systems — not demos, not prototypes, not tutorials.

This toolkit transforms Claude into a structured engineering collaborator capable of designing scalable architectures, ML systems, production frontend code, and coordinated multi-agent workflows.

---

## 🧠 Philosophy

Each skill in this toolkit is:

- **Opinionated** — Enforces best practices and flags anti-patterns  
- **Production-oriented** — Designed for deployable systems  
- **Structured** — Clear workflows, guardrails, and output formats  
- **Vendor-neutral** — No unnecessary framework coupling  
- **Composable** — Skills can be combined into full-stack workflows  

This is not prompt hacking.  
This is system design with discipline.

## 📁 Repository Structure

Each skill follows a consistent structure:

```
skill-name/
├── SKILL.md          # Core workflow, guardrails, output format
└── references/       # Deep-dive technical guidance
```

- `SKILL.md` is the entry point.
- `references/` contains advanced patterns loaded only when needed.

This progressive disclosure model keeps interactions focused while preserving depth.

---

## ⚙️ Usage

1. Open Claude → Settings → Custom Skills  
2. Import each skill directory  
3. Activate skills based on the task at hand  

Skills automatically trigger based on problem context (e.g., database design, ML system planning, multi-agent orchestration).

You can combine them:

- `db-architect` + `production-frontend`  
- `ml-solution-architect` + `multi-agent-orchestrator`  
- `pwa-builder` + `production-frontend`  
- All of them together for full-stack AI systems
